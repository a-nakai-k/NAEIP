# NAEIP
A MATLAB implementation of Nested Aggregation of Experts using Inducing Point (NAE-IP), an approximation of Gaussian process regression.

## Requirement
- [GPML MATLAB Code](http://www.gaussianprocess.org/gpml/code/matlab/doc/)

## Usage
Run on MATLAB 
```
main_NAEIP
```
